# ShelfCast Frontend

AI-driven stocking intelligence platform for retailers, manufacturers, and importers across West Africa.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4 + inline styles
- **Fonts:** Unbounded (headings), Poppins (body)

## Pages

- `/` — Landing page (Navbar, Hero, Features, How It Works, Pricing, CTA, Footer)
- `/auth` — 3-step onboarding flow (Email → Preferences → Payment)

## Getting Started

```bash
npm install
npm run dev
